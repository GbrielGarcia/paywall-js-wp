Paywall.js WP protege tu sitio WordPress cuando el cliente no paga, mostrando efectos visuales automáticos.

== Description ==

Integra Paywall.js en tu sitio WordPress para proteger tus pagos con efectos visuales automáticos. Permite configurar fecha de vencimiento, días de plazo, efectos visuales y colores desde el panel de administración.

== Installation ==

1. Sube la carpeta del plugin a `/wp-content/plugins/` o instala directamente desde el repositorio de WordPress.
2. Activa el plugin desde el menú 'Plugins' de WordPress.
3. Ve a Ajustes > Paywall.js para configurar las opciones.

== Frequently Asked Questions ==

= ¿Qué hace este plugin? =
Protege el contenido de tu sitio usando Paywall.js, mostrando efectos visuales cuando el pago ha vencido.

= ¿Puedo personalizar los efectos? =
Sí, puedes elegir entre gradiente, color sólido o fade, y definir los colores.
